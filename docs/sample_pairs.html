Sample pairs method¶

This module contains the required functions to implement Sample pairs attack method.

methods.sample_pairs.analyze(pix, h, w, channel='r')¶

Container analysis for steganography

Parameters

  • pix – Image pixels

  • h – Image height

  • w – Image width

  • channel – One of three possible RGB channels. Set ‘r’ for red, ‘g’ for green, ‘b’ for blue
