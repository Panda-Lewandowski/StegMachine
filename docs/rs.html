methods.rs.rs_test(img, bw=2, bh=2, mask=[1, 0, 0, 1])¶

Using the RS method.

This statistical method was by Jessica Friedrich, Miroslav Golyan and Andreas Pfitzman in 2001. It was named as the RS method. The original article can be found here: http://www.ws.binghamton.edu/fridrich/Research/acm_2001_03.pdf

Parameters

  • img – Image name

  • bw – Parameter to select the number of pixels in a row

  • bh – Parameter to select the number of pixels in a column

  • mask – Image mask for flipping
